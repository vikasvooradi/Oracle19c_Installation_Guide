  2. 2

    G:\oracle_19c\WINDOWS.X64_193000_db_home
  3. 3

05
System Tuning Performance parameters for Informatica workloads
SQL*Plus ¬∑ Performance Tuning 
-- Remove default password expiry
SQL> ALTER PROFILE DEFAULT LIMIT PASSWORD_LIFE_TIME UNLIMITED;
Profile altered.

-- Switch back to CDB root for system-level changes
SQL> ALTER SESSION SET CONTAINER=CDB$ROOT;
Session altered.

-- Increase open cursors for Informatica sessions
SQL> ALTER SYSTEM SET open_cursors = 5000 SCOPE=BOTH;
System altered.
‚úÖ Setting open_cursors = 5000 with SCOPE=BOTH applies the change to both the running instance and the spfile ‚Äî it persists across restarts.
